# Eve-Nantes


## Description :

Eve-Nantes est une application web evenementielle qui permet aux utilisateurs de découvrir, participer et organiser des sorties et des événements au sein de la ville de Nantes. L'application offre un design optimal sur lequel vous pouvez trouver des activités variées en fonction de vos mood du moment. 

Que vous soyez à la recherche d'une soirée animée, d'une activité culturelle ou d'une sortie en plein air, Eve-Nantes vous propose une sélection d'événements adaptés à vos envies.
